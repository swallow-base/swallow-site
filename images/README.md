# 画像フォルダ

このフォルダには、サイトで使用するすべての画像を配置します。

## ルート直下の共通画像

以下の画像をこのフォルダ（imagesルート）に直接配置してください：

### 必須
- `favicon.ico` - ブラウザタブ用アイコン（32×32px）
- `favicon.png` - Apple Touch Icon用（180×180px）
- `logo_3.png` - ヘッダーロゴ
- `logo_1b.png` - ヒーロー用大きいロゴ
- `ogp-default.jpg` - SNSシェア用画像（1200×630px）

## サブフォルダ

各ページ用の画像は、対応するサブフォルダに配置してください：

- `home/` - HOMEページ用画像
- `programs/` - プログラム関連画像
- `course/` - 受講プラン用画像
- `about/` - つばめ学習舎とは用画像
- `facility/` - 校舎紹介用画像
- `voice/` - 卒業生の声用画像
- `contact/` - お問い合わせ用画像
- `icons/` - アイコン類
- `designs/` - Canvaデザイン画像（参考用）

各サブフォルダには、配置すべき画像の詳細が記載されたREADME.mdがあります。

## 命名規則

### 基本ルール
- 英数字とハイフン（-）、アンダースコア（_）のみ使用
- 日本語や全角文字は使用しない
- 小文字推奨（既存ファイルlogo_3.png等は維持）
- スペース禁止

### ファイル形式
- **写真**: JPG（品質85%推奨）
- **透過が必要な画像**: PNG
- **アイコン**: SVG（推奨）

### ファイルサイズ目安
- ヒーロー画像: 300KB以下
- サムネイル: 100KB以下
- アイコン: 10KB以下

## HTMLからの参照例

```html
<!-- ルートページから -->
<img src="images/logo_3.png" alt="つばめ学習舎">
<img src="images/home/hero1.png" alt="ヒーロー背景">

<!-- programs/配下のページから -->
<img src="../images/logo_3.png" alt="つばめ学習舎">
<img src="../images/programs/coaching-hero.jpg" alt="コーチング">
```

## 注意事項

### 著作権・権利関係
- すべての画像は使用許可を得たもの、または自作のものを使用
- フリー素材を使用する場合は利用規約を確認

### 肖像権
- 人物が写っている写真は本人の許可を得て使用

### アクセシビリティ
- すべての画像に適切な `alt` 属性を設定
- 装飾画像の場合は `alt=""` で空に設定
