# JavaScript

このフォルダには、サイトで使用するJavaScriptを配置します。

## 配置予定のファイル

- `script.js` - サイト全体で使用する共通JavaScript

## 実装予定の機能

### モバイルメニュー
- ハンバーガーメニューの開閉
- メニューオーバーレイの表示/非表示

### スムーススクロール
- ページ内リンクのスムーズスクロール

### その他
- 必要に応じて追加

## 技術方針

- Vanilla JavaScript（jQuery不使用）
- ES6以降の記法を使用
- モダンブラウザ対応

## HTMLからの読み込み

```html
<script src="js/script.js"></script>

<!-- programs/配下のHTMLから -->
<script src="../js/script.js"></script>
```
