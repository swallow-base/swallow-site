# プログラムページ共通デザインガイド

他のプログラムページ（lecture.html, itsudemo.html, sparta.html, hitoritabi.html）作成時に引き継ぐ共通要素をまとめたドキュメントです。

---

## 1. 冒頭セクション（イントロセクション）

### 特徴
- **見出しに下線なし**（他セクションと差別化）
- モバイル：中央揃え + 下部に画像
- タブレット/PC：2カラムレイアウト（左テキスト・右画像）

### モバイル版（〜767px）

```css
.intro-section {
    background: white;
    padding: 60px 20px 0;
}

.intro-heading {
    font-size: 20px;
    font-weight: bold;
    color: var(--brand);
    text-align: center;
    margin-bottom: 28px;
    /* 下線なし - 他セクションと差別化 */
}

.intro-text {
    font-size: 15px;
    line-height: 2;
    color: #333;
    text-align: center;
}

.intro-image-wrapper {
    position: relative;
    width: 100vw;
    margin-left: calc(-50vw + 50%);
    margin-top: 30px;
    overflow: hidden;
    aspect-ratio: 5 / 2;  /* 縦:横 = 2:5 */
}

.intro-image-wrapper::before {
    content: "";
    position: absolute;
    top: -1px;  /* 上端の線を消す */
    left: 0;
    right: 0;
    height: 60%;
    background: linear-gradient(to bottom, 
        rgba(255, 255, 255, 1) 0%, 
        rgba(255, 255, 255, 1) 10%,  /* 10%まで完全な白 */
        rgba(255, 255, 255, 0) 100%
    );
    z-index: 1;
    pointer-events: none;
}

.intro-image {
    width: 100%;
    height: 100%;
    object-fit: cover;
    object-position: center;
    display: block;
}
```

## 2. CTAセクション（フッター直前）

### 基本ルール

| 画面サイズ | 表示 | 理由 |
|------------|------|------|
| モバイル（〜1023px） | **非表示** | フローティングCTAがあるため不要 |
| PC（1024px〜） | **表示** | フローティングCTAがないため必要 |

### 背景色の決め方

**直前セクションの背景色を引き継ぐ**

- 直前が「卒業生の声」（クリーム）→ CTAも **クリーム**
- 直前が「つばめの約束」（白）→ CTAも **白**

### CSS実装

```css
/* デフォルト：非表示 */
.cta-section {
    display: none;
}

.cta-button-large {
    display: inline-block;
    background: var(--accent);
    color: white;
    padding: 18px 40px;
    text-decoration: none;
    border-radius: 30px;
    font-size: 16px;
    font-weight: bold;
    box-shadow: 0 4px 12px rgba(225, 29, 47, 0.3);
    transition: transform 0.3s, box-shadow 0.3s;
}

.cta-button-large:hover {
    transform: translateY(-2px);
    box-shadow: 0 6px 16px rgba(225, 29, 47, 0.4);
}

/* PC版：表示 */
@media (min-width: 1024px) {
    .cta-section {
        display: block;
        background: var(--bg-cream);  /* または white（直前セクションに合わせる） */
        padding: 50px 20px;
        text-align: center;
    }
}
```

### HTML構造

```html
<!-- CTAセクション -->
<section class="cta-section">
    <a href="../contact.html" class="cta-button-large">体験入会・お問い合わせ</a>
</section>
```
