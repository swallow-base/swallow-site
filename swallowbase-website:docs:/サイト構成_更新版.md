## ナビゲーション順序

1. ホーム
2. プログラム（ドロップダウン）
   - プログラム一覧
   - コーチング
   - 集団指導・個別指導
   - LINE質問サービス
   - スパルタ・プログラム
   - ひとり旅プロジェクト
3. 受講プラン
4. つばめ学習舎とは
5. 校舎紹介
6. 卒業生の声
7. 体験入会・お問い合わせ（CTA）

## リンクパスの記述例

### ルートページから
```html
<a href="programs.html">プログラム</a>
<a href="programs/coaching.html">コーチング</a>
<a href="contact.html">お問い合わせ</a>
```

### programs配下のページから
```html
<a href="../index.html">ホーム</a>
<a href="../programs.html">プログラム一覧</a>
<a href="coaching.html">コーチング</a>
<a href="../contact.html">お問い合わせ</a>
```

### 画像パスの記述例

**ルートページ（index.html）から：**
```html
<img src="images/logo_3.png" alt="つばめ学習舎">
<img src="images/home/hero1.png" alt="ヒーロー背景">
<img src="images/programs/coaching-bg.jpg" alt="コーチング">
```

**programs配下のページから：**
```html
<img src="../images/logo_3.png" alt="つばめ学習舎">
<img src="../images/programs/coaching-hero.jpg" alt="コーチング">
```

## 共通要素

### ヘッダーナビゲーション
- 全ページ共通
- モバイル：ハンバーガーメニュー
- デスクトップ：横並びナビ + ドロップダウン

### フッター
- 全ページ共通
- 住所・連絡先情報
- 簡易ナビゲーション
- コピーライト

### CTA（Call To Action）
- 各ページに「体験入会・お問い合わせ」ボタン配置
- モバイル：フローティングボタン
- デスクトップ：セクション内に固定

## 画像管理

詳細は「画像管理リスト_最終版.md」を参照してください。

### 画像の配置ルール
- 共通画像：`images/` 直下
- ページ別画像：`images/ページ名/` に配置
- Canvaデザイン画像：`images/designs/` に配置（参考用）
- アイコン：`images/icons/` に配置

### 命名規則
- 英数字とハイフン、アンダースコアのみ
- 小文字に統一（既存ファイル除く）
- フォルダで分類するため、ファイル名はシンプルに
