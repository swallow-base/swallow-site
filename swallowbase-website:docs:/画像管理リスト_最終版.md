# つばめ学習舎 画像管理リスト

## 命名規則

### 基本ルール
- 英数字とハイフン、アンダースコアのみ使用
- 小文字に統一（既存ファイルで大文字が使われている場合は維持）
- フォルダで分類するため、ファイル名はシンプルに

### カテゴリ別プレフィックス
- `logo-` : ロゴ関連
- `hero-` : ヒーローセクション背景
- `design-` : Canvaデザイン画像（参考用）

| representative-02.jpg | 300×300px | 50～80KB | 荒井先生写真 | 正方形撮影<br>CSSで **丸く** トリミング（`border-radius: 50%`） |
