# つばめ学習舎 WEBサイト フォントサイズ規格

## 概要

全ページで統一して使用するフォントサイズの規格です。  
レスポンシブ対応として、以下のブレークポイントを設定しています。

- **モバイル**: 〜767px
- **タブレット**: 768px〜1023px
- **PC**: 1024px〜1199px（ナビゲーション切替、フローティングCTA非表示）
- **大画面**: 1200px〜

※フォントサイズは主に「モバイル / タブレット（768px〜） / 大画面（1200px〜）」の3段階で調整
※1024pxはナビゲーションやCTA表示切替に使用

## 使用例（CSS）

```css
/* モバイル（デフォルト） */
.section-heading {
    font-size: 20px;
}

.body-text {
    font-size: 15px;
}

.table-text {
    font-size: 14px;
}

/* タブレット（768px〜） */
@media (min-width: 768px) {
    .section-heading {
        font-size: 22px;
    }

    .body-text {
        font-size: 16px;
    }

    .table-text {
        font-size: 15px;
    }
}

/* PC（1024px〜）- 必要に応じて */
@media (min-width: 1024px) {
    /* ナビゲーションやCTA表示切替など */
}

/* 大画面（1200px〜） */
@media (min-width: 1200px) {
    .section-heading {
        font-size: 24px;
    }

    .body-text {
        font-size: 16px;
    }

    .table-text {
        font-size: 16px;
    }
}
```

## 備考

- line-height は原則 1.7〜1.9 を使用
- フォントファミリーは共通CSS（style.css）で定義済み
  - 日本語: Sawarabi Gothic
  - 英語: Hammersmith One
